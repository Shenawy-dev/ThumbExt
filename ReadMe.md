# YT-Thumb

This interface fetches the thumbnails of YouTube videos via the video link

## Install

`git clone https://github.com/Shenawy-devithub/ThumbExt.git`

## Usage

`node extr.js
Put Your Youtube Video: <VIDEO-LINK>`
